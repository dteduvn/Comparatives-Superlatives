📚 Grammar Guide
Lý thuyết so sánh hơn và so sánh nhất / Rules for Comparatives & Superlatives
💡 What are they? / Chúng là gì?
📊 Comparative – So sánh hơn
Used to compare 2 things / Dùng khi so sánh 2 vật/người
➤ My bag is heavier than yours.
🏆 Superlative – So sánh nhất
Used to say which is highest/lowest in a group / Dùng khi nói cái nhất trong nhóm
➤ This is the heaviest bag of all.
📏 Rules / Quy tắc
Short adjectives
Tính từ ngắn (1 âm tiết)
Add -er / -est

talltallertallest

CVC adjectives
Phụ+nguyên+phụ âm
Double last letter + -er / -est
bigbiggerbiggest

thinthinnerthinnest
wetwetterwettest
-y adjectives
Tính từ kết thúc -y
y → -ier / -iest

drydrierdriest

Tính từ dài (2+ âm tiết)

beautifulmore beautifulmost beautiful
